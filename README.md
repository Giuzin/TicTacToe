# TicTacToe Fullstack (Backend + Frontend)

Este projeto contém um **backend completo pronto para uso**, um **script SQL** e **arquivos auxiliares do frontend** (componentes e serviço Angular).

## Tecnologias utilizadas
- **Backend:** C# Core (.NET 8.0) + Entity Framework Core + Npgsql (PostgreSQL)  
- **Docker Compose:** Executa o banco de dados PostgreSQL e compila o container do backend  
- **Frontend:** Conjunto de arquivos Angular (componentes e serviço) prontos para incluir em um projeto Angular 15+
